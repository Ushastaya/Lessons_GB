# Подсказки по командной строке

Команда смены директории
```sh
cd d/GitGB
``````
Команда отображения текущей директории Linux, MaxOs
```sh
pwd
```
Листинг текущей директории 

Windows
```sh
dir
```
Linux, MaxOs
```sh
ls
```

Удаление файла

Windows
```sh
del <filename>
```
Linux, MaxOs
```sh
rm <filename>
```

Переход к другим состояниям

```sh
git checkout <comitname>
```
